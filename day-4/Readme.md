Control Flow / Execution Flow
- Code executed line-by-line
- Actions are performed & contol flow may change
    - Specific conditions can change contol flow
    - "if"
    - "else"
    - "else if"
    
- eg let a = 99;
     if a > 99 {
        println!("Big number")
     } else {
        println!("Small number")
     }
